The page class holds a chunk of data. This data can be operated on in various ways. Pages are stored in books and are deleted if they get too old.

#define PAGE_SIZE   100

The size of the data that this page holds. Since it is 2D, any page actually holds PAGE_SIZE*PAGE_SIZE pieces of data. Thus this should not get too large.
